# 📋 历史粘贴板

一个简洁轻量的 Windows 剪贴板历史管理工具，帮你记录和查找所有复制过的文字和图片。

---

## ✨ 主要功能

- 🔄 **自动记录**：后台静默监听，自动记录你复制的所有文字和图片
- 🗂️ **卡片展示**：时间降序排列，一目了然
- 🔍 **快速搜索**：输入关键词，秒找历史内容
- 📌 **置顶功能**：重要内容固定在顶部，不怕被淹没
- 🗑️ **一键删除**：不需要的记录随时清理
- ⏱️ **自动清理**：可设置 1/3/5 天保留期，过期自动删除
- 🔔 **轻提示**：复制后右下角弹出温柔提醒（可关闭）
- 🚀 **开机自启**：设置中一键开启，电脑开机自动运行
- 🎀 **淡粉色主题**：简洁优雅的视觉风格
- 🖼️ **文字+图片**：文字和截图都能记录

---

## 🚀 下载与使用

### 方式一：下载 .exe 直接运行（推荐）

1. 下载 `dist/历史粘贴板.exe`
2. 双击运行即可，**无需安装 Python**
3. 软件会出现在右下角系统托盘，单击图标打开面板

### 方式二：从源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行
python src/main.py
```

**环境要求**：Python 3.11+ / Windows 10+

---

## 🎮 使用说明

| 操作 | 方法 |
|------|------|
| 打开面板 | 单击右下角托盘图标 |
| 复制记录 | 左键点击卡片 |
| 置顶/删除 | 右键点击卡片 → 选择操作 |
| 搜索 | 在顶部搜索框输入关键词 |
| 设置 | 点击面板底部 ⚙ 按钮 |
| 退出 | 右键托盘图标 → 退出 |

| 技术 | 用途 |
|------|------|
| Python 3.11 | 开发语言 |

| SQLite | 本地数据存储 |
