# 📋 历史粘贴板

一个简洁轻量的 Windows 剪贴板历史管理工具，帮你记录和查找所有复制过的文字和图片。

---

## ✨ 主要功能

- 🔄 **自动记录**：后台静默监听，自动记录你复制的所有文字和图片
- 🗂️ **卡片展示**：时间降序排列，一目了然
- 🔍 **快速搜索**：输入关键词，秒找历史内容
- 📌 **置顶功能**：重要内容固定在顶部，不怕被淹没
- 🗑️ **一键删除**：不需要的记录随时清理
- ⏱️ **自动清理**：可设置 1/3/5 天保留期，过期自动删除
- 🎀 **淡粉色主题**：简洁优雅的视觉风格
