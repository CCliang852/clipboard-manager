## 1. 色彩方案

### 1.1 主色调：淡粉色系

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 (Primary) | `#FFB6C1` | 按钮、选中状态、强调元素 |
| 主色悬停 | `#FF9AAE` | 按钮悬停、交互反馈 |
| 主色浅色 | `#FFF0F3` | 卡片背景、选中卡片背景 |
| 背景色 | `#FFFAFB` | 窗口主背景 |
| 卡片背景 | `#FFFFFF` | 卡片底色 |
| 卡片阴影 | `rgba(255,182,193,0.15)` | 卡片投影 |
| 文字主色 | `#4A4A4A` | 正文文字 |
| 文字辅色 | `#9B9B9B` | 时间标签、辅助信息 |
| 分割线 | `#F5E6E8` | 卡片间分割 |
| 置顶标记 | `#FF8FA3` | 图钉图标颜色 |
| 搜索框边框 | `#FFD4DC` | 搜索框描边 |
| 删除按钮 | `#E88B8B` | 删除操作 |
| 滚动条 | `#FFD4DC` | 滚动条滑块 |

### 1.2 暗色模式
暂不支持，v1.0 仅浅色主题。

## 3. 主窗口布局

```
┌──────────────────────────────────┐
│  🔍  搜索剪贴板历史…              │  ← 搜索栏 (固定顶部, h=40px)
├──────────────────────────────────┤
│  ┌────────────────────────────┐  │
│  │ 📌 置顶卡片                 │  │  ← 置顶区 (如有)
│  │ "重要的文字内容…"           │  │
│  │ 2小时前                    │  │
│  └────────────────────────────┘  │
│  ┌────────────────────────────┐  │
│  │ 普通卡片                    │  │
│  │ "复制的文字内容…"           │  │  ← 滚动列表区
│  │ 5分钟前                    │  │     (时间降序)
│  └────────────────────────────┘  │
│  ┌────────────────────────────┐  │
│  │ 🖼️ 图片卡片                 │  │
│  │ [缩略图]                   │  │
│  │ 1小时前                    │  │
│  └────────────────────────────┘  │
│         …更多卡片…               │
├──────────────────────────────────┤
│  共 28 条记录    ⚙️ 设置        │  ← 底部状态栏 (h=30px)
└──────────────────────────────────┘
```

### 3.1 窗口规格
- 默认尺寸：380 × 600 像素
- 最小尺寸：300 × 400 像素
- 圆角窗口：8px 圆角（如果平台支持）
- 始终置顶：可选（默认关闭）

### 3.2 搜索栏
- 固定在列表顶部，不随滚动
- 左侧放大镜图标 + 输入框
- 输入时实时过滤，无结果时显示"无匹配记录"
- 清空搜索框恢复全部列表

## 4. 卡片组件规范

### 4.1 文字卡片
```
┌──────────────────────────────────┐
│ 📌                          ⋮   │  ← 置顶图标(条件) + 更多按钮
│                                  │
│ 这是一段复制的文字内容，         │
│ 超过三行会自动截断并显示         │
│ 省略号表示还有更多内容...        │
│                                  │
│ 3分钟前 · 来自 Chrome           │  ← 时间 + 来源(可选)
└──────────────────────────────────┘
```
- 内边距：12px
- 圆角：8px
- 最大行数：3 行，超出显示省略号
- 点击卡片 → 复制内容到剪贴板
- 右键卡片 → 菜单：复制 / 置顶(取消置顶) / 删除

### 4.2 图片卡片
```
┌──────────────────────────────────┐
│ 📌                          ⋮   │
│                                  │
│        ┌────────────┐           │
│        │            │           │  ← 缩略图 (宽填充, h=120px)
│        │  缩略图    │           │
│        │            │           │
│        └────────────┘           │
│                                  │
│ 1小时前                         │
└──────────────────────────────────┘
```
- 缩略图宽度撑满卡片，高度 120px，保持比例裁剪
- 点击卡片 → 复制图片到剪贴板

## 5. 交互规范

### 5.1 卡片点击
- 左键点击：复制内容到剪贴板 + 卡片短暂高亮（200ms 粉色闪烁）
- 右键点击：弹出上下文菜单

### 5.2 右键菜单
```
┌─────────────┐
│ 📋 复制      │
│ 📌 置顶      │
│ 🗑️ 删除      │
└─────────────┘
```

### 5.3 轻提示 (Toast)
- 位置：屏幕右下角
- 尺寸：240 × 50 像素
- 显示时长：1.5 秒
- 动画：从底部滑入 → 停留 → 淡出
- 样式：圆角 8px，粉色背景 `#FFB6C1`，白色文字
- 文案："已复制到历史记录 ✓"

### 5.4 搜索交互
- 输入文字即时过滤（300ms 防抖）
- 无匹配时显示空状态插画 + "没有找到相关内容"

### 5.5 删除确认
- 弹出简单确认框："确定删除这条记录吗？"
- 按钮：[取消] [删除]
