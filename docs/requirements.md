# 需求文档 — 历史粘贴板

> 版本：v1.0  
> 更新日期：2026-05-31  
> 状态：已确认，已开发完成

---

## 1. 项目背景与目标

### 1.1 背景
Windows 系统自带的剪贴板只能保留最后一次复制的内容，无法回溯历史。用户在日常工作中经常需要查找之前复制过的文字或图片，但缺乏一个轻量、直观的工具来管理剪贴板历史。

### 1.2 目标
开发一款 Windows 桌面软件，在后台自动记录用户复制的所有文字和图片，提供卡片式历史浏览、搜索、置顶和删除功能，以简洁直观的淡粉色 UI 呈现。

## 3. 功能清单

### 3.1 核心功能（必须实现）

#### F-01 剪贴板监听
- **描述**：后台持续监听 Windows 剪贴板变化
- **支持类型**：文字（纯文本）、图片（位图）
- **频率**：每 500ms 轮询一次
- **去重**：连续两次相同内容不重复记录
- **状态**：✅ 已确认

#### F-02 系统托盘
- **描述**：软件最小化到系统托盘，不占用任务栏空间
- **托盘菜单**：打开历史面板 / 设置 / 退出
- **交互**：单击托盘图标打开历史面板
- **状态**：✅ 已确认

#### F-03 历史面板（主窗口）
- **描述**：以卡片列表形式展示所有剪贴板记录
- **排序**：置顶项优先，其余按时间降序
- **窗口大小**：默认 380×600 像素，可调整
- **状态**：✅ 已确认

#### F-04 卡片组件
- **文字卡片**：显示文字内容（超过 3 行截断+省略号）
- **图片卡片**：显示缩略图
- **时间标签**：每条记录显示相对时间（如"3分钟前"）
- **状态**：✅ 已确认

#### F-05 搜索功能
- **描述**：顶部搜索框，输入文字实时筛选
- **匹配**：对文字记录进行包含匹配（忽略大小写）
- **图片**：搜索时不显示图片卡片（无法文本匹配）
- **状态**：✅ 已确认

#### F-06 置顶功能
- **描述**：右键卡片 → 选择"置顶"或"取消置顶"
- **表现**：置顶卡片始终在列表最上方，带视觉标记（如小图钉图标）
- **状态**：✅ 已确认

#### F-07 删除功能
- **描述**：右键卡片 → 选择"删除"，确认后移除
- **联动**：图片记录删除时同步删除本地图片文件
- **状态**：✅ 已确认

#### F-08 保留天数设置
- **描述**：设置面板中选择 1天 / 3天 / 5天
- **默认**：3天
- **清理**：每次启动时和每小时后自动清理过期记录
- **状态**：✅ 已确认

#### F-09 开机自启
- **描述**：通过 Windows 注册表或启动文件夹实现
- **可配置**：设置面板中可开关
- **状态**：✅ 已确认

#### F-10 复制轻提示
- **描述**：屏幕右下角短暂弹出半透明提示"已复制到历史记录"
- **时长**：1.5秒后自动消失
- **状态**：✅ 已确认

### 3.2 非功能需求

| 编号 | 类型 | 描述 |
|------|------|------|
| NF-01 | 性能 | 剪贴板监听不应造成 CPU 持续高占用 |
| NF-02 | 存储 | 图片应压缩存储，单张不超过 500KB |
| NF-03 | 稳定性 | 软件崩溃后重启不丢失历史数据 |
| NF-04 | 隐私 | 所有数据仅存储在本地，不上传网络 |
| NF-05 | 兼容性 | 支持 Windows 10 及以上系统 |
